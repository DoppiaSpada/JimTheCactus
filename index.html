My other applications

> PicturesExtractor (C for Linux)
> PicturesExtractor (C for Mac)

A command-line program able to randomly select the desired number of files - any type - from a specific directory and then copy them to another one.

First, compile the C source file on your machine. Second, invoke the program after compiling it, place your current directory on the folder containing the executable and write the next line on your shell with the right substitution: ./PicturesExtractor 'source_dir' 'dest_dir' 'number_of_files'.
